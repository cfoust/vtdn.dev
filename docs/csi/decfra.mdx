**DECFRA** (Fill Rectangular Area) is a CSI sequence that fills a rectangular screen region with a specified character.

## Syntax

```
CSI Pc ; Pt ; Pl ; Pb ; Pr $ x    DECFRA — Fill Rectangular Area
```

<FormalSyntax grammar={`

### Parameters

| Parameter | Description |
|-----------|-------------|
| `Pc` | Decimal character code to fill (e.g., 65 for 'A', 42 for '*') |
| `Pt` | Top row of the rectangle |
| `Pl` | Left column of the rectangle |
| `Pb` | Bottom row of the rectangle |
| `Pr` | Right column of the rectangle |

## Description

Per the VT510 manual, DECFRA fills the specified rectangular area with the character whose code is `Pc`. The current SGR attributes are applied to the filled characters. The cursor position is not changed.

The `Pc` value must be a printable character code (32-126 for ASCII, or an appropriate value in the active character set). If `Pc` is not a printable character, the result is undefined.

Coordinates are 1-based and refer to absolute screen positions.

## Examples

```bash
printf '\e[42;1;1;24;80$x'    # Fill entire 80x24 screen with '*' (42)
printf '\e[65;5;10;15;30$x'   # Fill rectangle (5,10)-(15,30) with 'A' (65)
```
