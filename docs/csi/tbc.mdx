**TBC** (Tabulation Clear) is a CSI sequence that clears one or more tab stops.

## Syntax

```
CSI Ps g      TBC — Tabulation Clear
```

<FormalSyntax grammar={`
TBC = 0x1b, "[", [ Ps ], "g" ;
Ps  = "0" | "3" ;                       (* default: 0 *)
`} />

### Parameters

| Value | Meaning |
|-------|---------|
| `0` | Clear the tab stop at the current column (default) |
| `3` | Clear all tab stops |

## Description

Per ECMA-48 §8.3.154, TBC clears tabulation stops depending on the parameter value. The full standard defines values 0-5, but terminal emulators commonly support only two:

- **`CSI 0 g`** or **`CSI g`** — Clears the tab stop at the current cursor column.
- **`CSI 3 g`** — Clears all character tab stops.

After clearing all tab stops, HT (0x09) will have no effect until new stops are set with HTS.

## Examples

```bash
printf '\e[g'        # Clear tab stop at current column
printf '\e[3g'       # Clear all tab stops
```
