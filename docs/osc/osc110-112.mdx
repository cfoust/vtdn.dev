**OSC 110**, **OSC 111**, and **OSC 112** reset the terminal's default foreground color, background color, and cursor color to their configured defaults. They are the counterparts to OSC 10, OSC 11, and OSC 12 respectively.

## Syntax

```
OSC 110 ST       Reset default foreground color
OSC 111 ST       Reset default background color
OSC 112 ST       Reset cursor color
```

<FormalSyntax grammar={`
OSC110 = 0x1b, "]", "110", ( 0x07 | 0x1b, "\\\\" ) ;
OSC111 = 0x1b, "]", "111", ( 0x07 | 0x1b, "\\\\" ) ;
OSC112 = 0x1b, "]", "112", ( 0x07 | 0x1b, "\\\\" ) ;
`} />

## Description

These sequences undo color changes made by their corresponding OSC set/query sequences:

| Sequence | Resets | Counterpart |
|----------|--------|-------------|
| OSC 110 | Default foreground (text) color | OSC 10 |
| OSC 111 | Default background color | OSC 11 |
| OSC 112 | Cursor color | OSC 12 |

Each reset sequence takes no parameters. The color reverts to whatever the terminal's configuration specifies as the default. This is the color that was in effect before any OSC 10/11/12 sequences were processed.

These sequences are particularly useful for applications that temporarily change the terminal's appearance (for example, to implement a dark/light theme toggle) and need to restore the original colors when the application exits.

:::note
There are additional reset sequences in the same range: OSC 113 resets the mouse foreground color, OSC 114 resets the mouse background color, OSC 115 resets the Tektronix foreground, OSC 116 resets the Tektronix background, and OSC 117 resets the highlight color. However, support for these is rare outside of XTerm.
:::

## Examples

```bash
# Change foreground to red, then reset it
printf '\e]10;#ff0000\e\\'
printf '\e]110\e\\'

# Change background to blue, then reset it
printf '\e]11;#0000ff\e\\'
printf '\e]111\e\\'

# Change cursor color to green, then reset it
printf '\e]12;#00ff00\e\\'
printf '\e]112\e\\'

# Reset all three at once
printf '\e]110\e\\' && printf '\e]111\e\\' && printf '\e]112\e\\'
```
