**Standard ANSI colors** use SGR parameters to set the foreground and background to one of 8 normal or 8 bright colors.

## Syntax

```
CSI 30-37 m       SGR — Set foreground to standard color
CSI 39 m          SGR — Reset foreground to default
CSI 40-47 m       SGR — Set background to standard color
CSI 49 m          SGR — Reset background to default
CSI 90-97 m       SGR — Set foreground to bright color
CSI 100-107 m     SGR — Set background to bright color
```

### Parameters

**Foreground colors (30-37):**

| Code | Color |
|------|-------|
| `30` | Black |
| `31` | Red |
| `32` | Green |
| `33` | Yellow |
| `34` | Blue |
| `35` | Magenta |
| `36` | Cyan |
| `37` | White |
| `39` | Default foreground |

**Background colors (40-47):**

| Code | Color |
|------|-------|
| `40`-`47` | Same colors as 30-37 |
| `49` | Default background |

**Bright colors (90-97, 100-107):**

The bright/intense color variants. Codes 90-97 set bright foreground; 100-107 set bright background.

## Description

Per ECMA-48 §8.3.117, SGR parameters 30-37 set the foreground display color and 40-47 set the background color. ECMA-48 defines these as: black (30/40), red (31/41), green (32/42), yellow (33/43), blue (34/44), magenta (35/45), cyan (36/46), and white (37/47).

The bright color ranges (90-97 for foreground, 100-107 for background) are not part of ECMA-48 but are a widely supported extension originally from AIXTerm. They provide access to the "bright" or "intense" versions of the 8 standard colors without requiring the bold attribute.

SGR 39 and 49 reset the foreground and background to their default colors respectively. SGR 0 resets all colors and attributes.

The actual RGB values for these colors are not standardized and vary between terminals. Most terminals allow users to customize them.

## Examples

```bash
printf '\e[31mRed text\e[0m\n'
printf '\e[32;43mGreen on yellow\e[0m\n'
printf '\e[91mBright red text\e[0m\n'
printf '\e[97;104mBright white on bright blue\e[0m\n'
```
