**24-bit truecolor (semicolons)** uses SGR 38;2 and 48;2 with semicolon-separated RGB values to set foreground and background to any of 16.7 million colors.

## Syntax

```
CSI 38;2;r;g;b m    SGR — Set foreground to RGB color
CSI 48;2;r;g;b m    SGR — Set background to RGB color
```

### Parameters

| Parameter | Description |
|-----------|-------------|
| `r` | Red component (0-255) |
| `g` | Green component (0-255) |
| `b` | Blue component (0-255) |

## Description

The semicolon-separated truecolor syntax is the most widely supported form of 24-bit RGB color in terminals. It extends the ECMA-48 SGR 38/48 color mechanism with `2` as the color space indicator, followed by the red, green, and blue components.

This syntax deviates from the ITU-T T.416 standard, which specifies colons as sub-parameter separators (see the colon variant). However, the semicolon form was adopted first by terminal emulators and has broader support.

Colors set via RGB are not affected by palette changes and always represent the exact specified color.

## Examples

```bash
printf '\e[38;2;255;100;0mOrange text\e[0m\n'
printf '\e[48;2;0;0;128mNavy background\e[0m\n'
printf '\e[38;2;255;255;255;48;2;0;0;0mWhite on black\e[0m\n'
```
