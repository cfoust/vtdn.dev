**Underline color (RGB)** uses SGR 58;2 to set the underline color to an arbitrary 24-bit RGB value, independently of the foreground color.

## Syntax

```
CSI 58;2;r;g;b m    SGR — Set underline color to RGB
CSI 59 m            SGR — Reset underline color to default
```

### Parameters

| Parameter | Description |
|-----------|-------------|
| `r` | Red component (0-255) |
| `g` | Green component (0-255) |
| `b` | Blue component (0-255) |

## Description

SGR 58;2 sets the underline color to an RGB value using the same syntax as SGR 38;2 (foreground RGB), but applied to the underline decoration. This allows precise color control of underline decorations independently of the text color.

SGR 59 resets the underline color to the default (typically the foreground color).

The colon-separated form `SGR 58:2::r:g:b` is also valid (per ITU-T T.416) and is accepted by some terminals.

## Examples

```bash
# Red curly underline with RGB color
printf '\e[4:3;58;2;255;0;0mError here\e[0m\n'

# Orange underline on normal text
printf '\e[4;58;2;255;165;0mWarning\e[0m\n'
```
